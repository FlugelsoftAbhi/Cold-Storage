Production Ingestion Setup Guide (Google Form & GitHub Actions)

1
Google Form & Sheet

Create a Google Form with fields: Warehouse Name, Owner, Email, Phone, City, State, Type, Capacity, Vacant, Price, Amenities. Set responses to output to a Google Sheet.

2
Publish Responses

In the Google Sheet responses tab, go to File > Share > Publish to web. Under Link options, choose the responses sheet and format as Comma-separated values (.csv).

3
Set Action Secrets

Copy the published CSV URL. In your GitHub Repository settings, navigate to Secrets and variables > Actions and create a secret named GOOGLE_SHEET_CSV_URL with your URL.

4
Automated Action

A daily GitHub Action running update_dashboard.yml runs the Python index compiler to fetch sheet updates, de-duplicate official and crowdsourced nodes, and update the directory index.
